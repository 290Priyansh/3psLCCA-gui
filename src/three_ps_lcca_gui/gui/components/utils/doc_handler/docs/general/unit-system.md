# Unit System

The measurement system applied to all dimensional inputs in this project.

## Options

| Option | Length | Mass | Volume |
| :--- | :--- | :--- | :--- |
| **SI (Metric)** | Metres (m) | Kilograms (kg) | Cubic metres (m³) |
| **Imperial** | Feet (ft) | Pounds (lb) | Cubic feet (ft³) |
